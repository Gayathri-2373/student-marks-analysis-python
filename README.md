# Student Marks Analysis (Python)

## Description
This project analyzes student marks using Python and pandas.

## Features
- Load CSV data
- Calculate average marks
- Identify highest scorer
- Visualize data using bar chart

## Technologies Used
- Python
- pandas
- matplotlib

## How to Run
1. Install Python
2. Install libraries:
   pip install pandas matplotlib
3. Run:
   python analysis.py
